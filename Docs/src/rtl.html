RTL Version

In order to have RTL mode enabled, uncomment Light Theme scss files in the src/assets/theme.scss file same as uncomment below 2 line 

// @import "./bootstrap.scss";
// @import "./app.scss";
and uncomment RTL Mode scss files in the src/assets/theme.scss file same as uncomment below lines 

// @import "./rtl/bootstrap-rtl";
// @import "./rtl/components-rtl";
// @import "./rtl/float-rtl";
// @import "./rtl/general-rtl";
// @import "./rtl/pages-rtl";
// @import "./rtl/plugins-rtl";
// @import "./rtl/spacing-rtl";
// @import "./rtl/structure-rtl";
// @import "./rtl/text-rtl";
and comment Dark Theme scss files in the src/assets/theme.scss file.

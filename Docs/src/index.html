Introduction

To use most of the Doot build tools, Node.js LTS version is required. Some of the plugins and framework in Doot does not yet support the latest Node.js version.

Doot is a fully featured premium chat app template in React Redux Saga with Firebase/Fake-backend with Facebook and Google Authentication and developer-friendly codes. We have not used jQuery in this template its pure ReactJs admin template with components based only. it's created with Typescript.

Google and Facebook Authentication is not saved in Firebase/Fake-backend. Its just working with local storage.

Doot is a chat application template that is a beautifully crafted, clean & minimal designed admin template with Dark, Light Modes.

It will help your team moving faster and saving development costs and valuable time.
